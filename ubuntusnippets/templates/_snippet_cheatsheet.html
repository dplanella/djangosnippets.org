Markdown cheatsheet

Bold and italic text:
**bold** and *italic* text
Links:
[link text](link URL)
Code:
`code`
Code block:
Indent each line four spaces
Block quote:
Start each line with >
Unordered list:
Start each item with *
Ordered list:
Start each item with a number and a period: 1.
